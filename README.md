# Anu Hospitality Staff Website

This project is the official business website developed for **Anu Hospitality Staff Pvt. Ltd.**, a professional hospitality and domestic cleaning service provider based in London.

The website helps the company establish a strong online presence and allows customers to easily explore services, contact the company, and book domestic cleaning services online.

## Features

- Modern and responsive user interface
- Online booking system for domestic cleaning services
- Secure online payments using **Stripe**
- Contact form for customer inquiries
- Admin dashboard to manage bookings and website data
- Database storage for customers and bookings

## Tech Stack

Frontend
- HTML5
- Tailwind CSS
- JavaScript
- AngularJS

Backend
- Laravel

Database
- MySQL

Payment Integration
- Stripe

## Functionality

Customers can browse available services and book domestic cleaning services directly through the website. After booking, the payment is securely processed using **Stripe**, and the booking details are stored in the database.

The admin panel allows the administrator to:

- View all bookings
- Manage customer data
- Monitor website activity
- Manage overall website functionality
